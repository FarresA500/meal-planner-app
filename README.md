# 🥗 AI Meal Planner

🚀 **AI Meal Planner** is a smart meal planning app that generates a **7-day meal plan** based on your **daily calorie goals**.  
It provides **meal names, ingredients, preparation steps, and estimated grocery costs**.

## 🌟 Features
✅ Generate a **weekly meal plan** based on calorie input  
✅ Shows **meal names, ingredients, and recipes**  
✅ Connects with **real grocery prices from online stores** (Coming Soon!)  
✅ Clean and **mobile-friendly UI**

## 🛠️ How It Works
1. **Enter your daily calorie goal** (e.g., 2200 calories).  
2. Click **"Generate Plan"**.  
3. AI will create a **7-day meal plan** with:  
   - 🥞 **Breakfast**
   - 🍎 **Snacks**
   - 🥗 **Lunch**
   - 🍽 **Dinner**  
4. Future updates will include **real-time grocery price comparisons**.

## ⚙️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Flask (Python) (Hosted on Replit)  
- **Hosting:** Vercel (Frontend) & Replit (Backend)  
- **APIs Used:** OpenAI GPT for meal generation
